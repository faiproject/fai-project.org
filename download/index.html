Package download

The FAI software is divided in several packages:
  • fai-server: Package needed on the install server
  • fai-client: This package is needed on all hosts
  • fai-doc: The documentation
  • fai-nfsroot: Files only needed inside the nfsroot
  • fai-setup-storage: Contains the setup-storage tool
  • fai-quickstart: Meta package for easy setup of an install server

Entry for /etc/apt/sources.list and for /etc/fai/apt/sources.list: 


deb https://fai-project.org/download bullseye koeln
The packages are signed with the gpg key of Thomas Lange. Add the gpg key in your apt configuration: 

$ wget https://fai-project.org/download/fai-project.gpg
$ sudo cp fai-project.gpg /etc/apt/trusted.gpg.d/

FAI packages from the official project repository or from a Debian mirror.

Packages for Ubuntu

Due to a bug in Ubuntu, you cannot create an Ubuntu nfsroot. See https://bugs.launchpad.net/ubuntu/+source/linux/+bug/1660587. On an FAI server running Ubuntu, you must create a Debian nfsroot and also download an Ubuntu base file and put this into the basefiles subdirectory.
  • Official Ubuntu packages
  • Ubuntu PPA.
  • Sources.list entry for Ubuntu packages: 
    deb http://archive.ubuntu.com/ubuntu/ jammy universe
