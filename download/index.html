FAI - Fully Automatic Installation

Package download

The FAI software is divided in several packages:
  • fai-server: Package needed on the install server
  • fai-client: This package is needed on all hosts
  • fai-doc: The documentation
  • fai-nfsroot: Files only needed inside the nfsroot
  • fai-setup-storage: Contains the setup-storage tool
  • fai-quickstart: Meta package for easy setup of an install server
  • Entry for /etc/apt/sources.list and for /etc/fai/apt/sources.list: 

    deb https://fai-project.org/download jessie koeln

    or 

    deb https://fai-project.org/download stretch koeln

  • The packages are signed with the gpg key of Thomas Lange. Two ways of getting the gpg key in your apt configuration 
    gpg -a --recv-keys 2BF8D9FE074BCDE4; gpg -a --export 2BF8D9FE074BCDE4 | apt-key add -
    wget -O - https://fai-project.org/download/2BF8D9FE074BCDE4.asc | sudo apt-key add -

  • FAI packages from the official project repository or from a Debian mirror.

Packages for Ubuntu

Up-to-date official packages: 

deb http://archive.ubuntu.com/ubuntu/ xenial universe

PPA containing FAI 5.0 and older versions: 

deb http://ppa.launchpad.net/fai/ppa/ubuntu wily main

Base files for Ubuntu, CentOS and Scientific Linux Cern

  • Base files are needed for Ubuntu, CentOS or SLC if your nfsroot is a Debian one.
