API documentation of the FAI.me service

String variables:
email
username
userpw
rootpw
addpkgs      list of packages space separated
github       github user for ssh root access
repo         same format as in sources.list
String variables with two values (undefined or the uppercase string listed here)
cl1   BACKPORTS              cl5   SSH_SERVER
cl2   DEBIAN_DEVEL           cl6   STANDARD
cl3   WEBSERVER              cl7   NONFREE
cl4   PRINTSERVER
Select menues define these variables:
partition    ONE|ONE_LVM|HOME|HOME_LVM
desktop      <empty>|LXDE|KDE|MATE|CINNAMON|XFCE|GNOME
keyboard     gb|us|fr|de|it|pl|pt|ru|es|tr|cn|tw|hi|fi|ja
suite        buster|bullseye|bookworm
type	     install|cloud
Variables used only when type=cloud
disksize     int <= 100
format       raw.xz|raw.zst|qcow2|vdi|vhdx|vmdk
hostname     string
Examples of URL's
https://fai-project.org/cgi/faime.cgi?type=install;username=debian;partition=ONE;keyboard=us;suite=stretch;cl1=BACKPORTS;cl5=SSH_SERVER;cl6=STANDARD;cl7=NONFREE;sbm=2

https://fai-project.org/cgi/faime.cgi?type=install;username=debian;userpw=fai;github=mrfai;partition=HOME_LVM;keyboard=de;suite=buster;desktop=XFCE;cl5=SSH_SERVER;cl6=STANDARD;cl7=NONFREE;addpkgs=mc%20gimp;email=you%40yourmailserver.org;sbm=2

https://fai-project.org/cgi/faime.cgi?type=cloud;disksize=5;format=raw.zst;hostname=debian;rootpw=fai;username=debian;userpw=fai;keyboard=us;suite=buster;cl5=SSH_SERVER;cl6=STANDARD;cl7=NONFREE;addpkgs=zile;sbm=2
