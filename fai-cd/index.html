FAI CD: The Easy Way to Start With FAI

downloadFAI ISO large (1.1GB)

The default image. Includes all packages for XFCE and GNOME but only the basefiles for CentOS 7 and Ubuntu 18.04

downloadFAI ISO small (670MB)

Only the XFCE packages are included. Other packages and basefiles are downloaded from the internet.

downloadFAI ISO Ubuntu (1.2GB)

This image only installs Ubuntu 20.04 LTS. All packages are included.

downloadAutodiscover CD (31MB)

Boot only image which autodiscovers the FAI server.

All installation images use Debian 10 aka buster, FAI 5.9.4, kernel 4.19, amd64 architecture, sha256sum.asc of the ISO's. The can boot from CD/DVD or USB stick with legacy BIOS or UEFI.

If none of the ISOs fit your needs, use the FAI.ME build service. This also supports Debian 9 and 10 with a backports kernel.

FAI on a USB stick
You can write the ISO image onto a USB stick using the command dd(1): 
  dd if=faicd64-large_5.9.iso of=/dev/sdx
and replace /dev/sdX with the name of your USB device (e.g. /dev/sdf).
