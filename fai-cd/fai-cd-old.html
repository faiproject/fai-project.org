FAI CD: The Easy Way to Start With FAI

downloadFAI ISO large (1.8GB)

The default image. Includes all packages for the XFCE and GNOME desktop but only the basefiles for Rocky Linux 9 and Ubuntu 22.04

downloadFAI ISO small (875MB)

Includes packages for a stripped down XFCE desktop (without libreoffice). Other packages and basefiles are downloaded from the internet.

downloadFAI ISO Ubuntu (1.7GB)

This image only installs Ubuntu 22.04 LTS. All packages are included.

downloadAutodiscover CD (49MB)

Boot only image which autodiscovers the FAI server.

All installation images use Debian 12 aka bookworm, FAI 6.0.3, kernel 6.1, amd64 architecture, sha256sum.asc of the ISO's. They can boot from CD/DVD or USB stick with legacy BIOS or UEFI.

If none of the ISOs fit your needs, use the FAI.ME build service. This also supports Debian 10 with a backports kernel.

FAI on a USB stick
You can write the ISO image onto a USB stick using the command dd(1) and replace /dev/sdX with the name of your USB device (e.g. /dev/sdf): 

dd if=faicd64-large_6.0.3.iso of=/dev/sdx
